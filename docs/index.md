# Calendar Viewer for Luma Events

**Show your [Lu.ma](https://lu.ma) events beautifully on your WordPress site.**

Bring your organization's Lu.ma calendar into WordPress with polished,
ready-made views you can drop onto any page — as a block, an Elementor widget,
or a shortcode.

## ⬇️ Get it

Download **[luma-viewer.zip](https://github.com/sveinmagnus/LumaViewer/releases/latest/download/luma-viewer.zip)**,
then in WordPress go to **Plugins → Add New → Upload Plugin**. See the
**[installation guide](installation.md)** for the full walkthrough.

## ✨ What you get

- 📅 **Six beautiful views** — List, Week, Month, Day, Photo, and Summary — with
  on-page switching and date browsing.
- 🧩 **Works with your editor** — Gutenberg block, Elementor widget, and
  shortcode, all with the same options.
- 🎨 **Lists, your way** — full cards, compact rows, or minimal; grouped by day
  or month. Perfect for a main page or a sidebar.
- 🔒 **Members-only events** — with MemberPress, gate events by category; others
  see a teaser or nothing.
- ⚡ **Fast and always fresh** — cached and refreshed in the background.
- 🔎 **Search-engine friendly** — each event gets its own page with rich event
  data built in.
- 🔗 **One-click register** — buttons link straight to the event on Lu.ma.
- 🔄 **Automatic updates** — new versions appear right in your dashboard.
